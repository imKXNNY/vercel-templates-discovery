# Semantic Search Implementation Plan — Issue #11

> **For Hermes:** Use parallel subagents or direct execution task-by-task. This plan assumes the implementer has zero context for the codebase.

**Goal:** Add local semantic search to the Vercel Templates catalog so users can search by intent ("ecommerce chatbot") instead of matching exact keywords.

**Architecture:** Keep embeddings and vector search inside the existing SQLite cache. The Ollama local API (`nomic-embed-text:latest`) encodes template text into 768-d vectors; vectors are stored in a `sqlite-vec` virtual table and searched with cosine similarity. Both Python and TypeScript runtimes get parallel, parity implementations: Python first, TypeScript second. MCP and REST surfaces gain a `semantic` query option.

**Tech Stack:**
- Embeddings: Ollama local API (`nomic-embed-text:latest`, 768 dims) via HTTP
- Vector store: `sqlite-vec` inside the existing SQLite cache
- Python runtime: `requests` (already a dependency) + `sqlite-vec` + `numpy` (for vector math)
- TypeScript runtime: `fetch` calling Ollama + `sqlite-vec` via `better-sqlite3` extension
- CI / disconnected environments: deterministic `FakeEmbeddingModel` that produces normalized 768-dim vectors
- Optional offline fallback: `sentence-transformers` / `@xenova/transformers` if Ollama is not reachable and the optional extra is installed

## Design decisions

1. **Vector store stays in SQLite.** Avoids introducing a separate service and keeps the tool portable.
2. **Embedding text per template:** `title + description + frameworks + use_cases + css + databases + authentication + cms + readme_text[:2000]`.
3. **Model choice:** Use Ollama's local `nomic-embed-text:latest` model (768 dims, ~274 MB, already downloaded). Default `VTD_OLLAMA_URL` to `http://localhost:11434/api/embed`. Accept `VTD_EMBEDDING_MODEL` to override the model name. If Ollama is unreachable, fall back to the `FakeEmbeddingModel` (deterministic) for tests and optionally to `sentence-transformers`/`@xenova/transformers` if the optional extra is installed.
4. **Distance metric:** cosine similarity (`sqlite-vec` `vec_distance_cosine`).
5. **No re-train on every `index`:** compute embeddings at index time and cache. On subsequent `index`, only re-encode if source text changed. For simplicity in the first version, always re-encode on `index` (the dataset is small; ~300 templates * 768 dims = ~1 MB). Optimize later if needed.
6. **Python implementation first, TypeScript second** to validate the UX before porting.
7. **Optional extras:** add a new `semantic` extra in `pyproject.toml` for `sqlite-vec` and `numpy` only (no heavy ML library). Keep Ollama as the default embedding provider; `sentence-transformers` remains an optional offline fallback.
8. **FTS5 fallback:** if `sqlite-vec` is not installed or Ollama is unreachable, `search` still works via FTS5; `semantic` query fails gracefully with a clear message.
9. **MCP/REST parity:** add `search_templates_semantic(query, limit)` tool and `GET /templates/semantic?q=...` endpoint.
10. **Tests with mocked embeddings:** avoid calling real embedding APIs in CI. Use deterministic `FakeEmbeddingModel` producing 768-dim normalized vectors and verify vector search math.

```python
from __future__ import annotations

import os
from typing import Any

import numpy as np

_OLLAMA_URL = os.getenv("VTD_OLLAMA_URL", "http://localhost:11434/api/embed")
_MODEL_NAME = os.getenv("VTD_EMBEDDING_MODEL", "nomic-embed-text:latest")
_DIMENSIONS = 768


class EmbeddingModel:
    """Abstract embedding model interface."""

    def __init__(self, model_name: str = _MODEL_NAME, dimensions: int = _DIMENSIONS) -> None:
        self.model_name = model_name
        self.dimensions = dimensions

    def encode(self, texts: list[str]) -> np.ndarray:
        raise NotImplementedError

    def encode_single(self, text: str) -> np.ndarray:
        return self.encode([text])[0]


class OllamaEmbeddingModel(EmbeddingModel):
    """Embedding model backed by a local Ollama API."""

    def __init__(
        self,
        model_name: str = _MODEL_NAME,
        dimensions: int = _DIMENSIONS,
        ollama_url: str = _OLLAMA_URL,
        timeout: float = 30.0,
    ) -> None:
        super().__init__(model_name, dimensions)
        self.ollama_url = ollama_url
        self.timeout = timeout

    def encode(self, texts: list[str]) -> np.ndarray:
        import requests

        resp = requests.post(
            self.ollama_url,
            json={"model": self.model_name, "input": texts},
            timeout=self.timeout,
        )
        resp.raise_for_status()
        data = resp.json()
        embeddings = data.get("embeddings", [])
        if not embeddings:
            raise RuntimeError("Ollama returned no embeddings")
        vectors = np.asarray(embeddings, dtype=np.float32)
        if vectors.shape[1] != self.dimensions:
            self.dimensions = vectors.shape[1]
        return vectors


class FakeEmbeddingModel(EmbeddingModel):
    """Deterministic fake model for tests and CI."""

    def __init__(self, dimensions: int = _DIMENSIONS) -> None:
        super().__init__(model_name="fake", dimensions=dimensions)

    def _vector_for(self, text: str) -> np.ndarray:
        rng = np.random.default_rng(hash(text) % 2**32)
        vec = rng.random(self.dimensions).astype(np.float32)
        vec /= np.linalg.norm(vec)
        return vec

    def encode(self, texts: list[str]) -> np.ndarray:
        return np.array([self._vector_for(t) for t in texts], dtype=np.float32)


class SentenceTransformersEmbeddingModel(EmbeddingModel):
    """Offline fallback using sentence-transformers (optional extra)."""

    def __init__(self, model_name: str = "sentence-transformers/all-MiniLM-L6-v2") -> None:
        from sentence_transformers import SentenceTransformer

        super().__init__(model_name=model_name, dimensions=384)
        self._model = SentenceTransformer(self.model_name, device="cpu")

    def encode(self, texts: list[str]) -> np.ndarray:
        embeddings = self._model.encode(texts, convert_to_numpy=True, normalize_embeddings=True)
        return np.asarray(embeddings, dtype=np.float32)


def get_model(fake: bool = False) -> EmbeddingModel:
    if fake:
        return FakeEmbeddingModel()
    try:
        return OllamaEmbeddingModel()
    except Exception:
        # Optional: try sentence-transformers fallback here if importable.
        return FakeEmbeddingModel()


def embedding_text(template: dict[str, Any]) -> str:
    parts = [
        template.get("title", ""),
        template.get("description", ""),
        template.get("frameworks", ""),
        template.get("use_cases", ""),
        template.get("css", ""),
        template.get("databases", ""),
        template.get("authentication", ""),
        template.get("cms", ""),
    ]
    readme = template.get("readme_text", "")[:2000]
    if readme:
        parts.append(readme)
    return " ".join(p for p in parts if p).strip()
```

```python
def semantic_search(self, query: str, limit: int = 10) -> list[dict[str, Any]]:
    if self.embedding_model is None:
        raise RuntimeError(
            "Semantic search requires the semantic extra. Install with: "
            "pip install 'vercel-templates-discovery[semantic]'"
        )
    conn = sqlite3.connect(self.db_path)
    conn.enable_load_extension(True)
    try:
        import sqlite_vec
        sqlite_vec.load(conn)
    except Exception as exc:
        conn.close()
        raise RuntimeError("sqlite-vec extension not available") from exc

    conn.row_factory = sqlite3.Row
    query_vec = self.embedding_model.encode_single(query)
    cursor = conn.execute(
        """
        SELECT t.*, distance
        FROM embeddings e
        JOIN templates t ON t.id = e.rowid
        WHERE e.embedding MATCH ?
        ORDER BY distance
        LIMIT ?
        """,
        (query_vec.tobytes(), limit),
    )
    rows = [dict(row) for row in cursor.fetchall()]
    conn.close()
    return rows
```

Note: verify `MATCH` syntax for sqlite-vec; alternative may be `vec_distance_cosine(e.embedding, vec_f32(?))`.

```typescript
import type { TemplateInput } from "./types.js";

export const DEFAULT_MODEL = "Xenova/all-MiniLM-L6-v2";
export const DIMENSIONS = 384;

export interface EmbeddingModel {
    encode(texts: string[]): Promise<Float32Array[]>;
    encodeSingle(text: string): Promise<Float32Array>;
}

export class FakeEmbeddingModel implements EmbeddingModel {
    constructor(private dimensions: number = DIMENSIONS) {}

    encode(texts: string[]): Float32Array[] {
        return texts.map((t) => this.encodeSingle(t));
    }

    encodeSingle(text: string): Float32Array {
        const buf = new Uint32Array(1);
        buf[0] = hashCode(text) >>> 0;
        const rng = mulberry32(buf[0]);
        const vec = new Float32Array(this.dimensions);
        let sum = 0;
        for (let i = 0; i < this.dimensions; i++) {
            const v = rng();
            vec[i] = v;
            sum += v * v;
        }
        const norm = Math.sqrt(sum);
        for (let i = 0; i < this.dimensions; i++) vec[i] /= norm;
        return vec;
    }
}

export class SentenceTransformersEmbeddingModel implements EmbeddingModel {
    private pipeline: any | null = null;
    constructor(private modelName: string = DEFAULT_MODEL) {}

    async encode(texts: string[]): Promise<Float32Array[]> {
        const { pipeline } = await import("@xenova/transformers");
        if (!this.pipeline) {
            this.pipeline = await pipeline("feature-extraction", this.modelName);
        }
        const output = await this.pipeline(texts, { pooling: "mean", normalize: true });
        return output.map((o: any) => new Float32Array(o.data));
    }

    async encodeSingle(text: string): Promise<Float32Array> {
        return (await this.encode([text]))[0];
    }
}

export function embeddingText(t: TemplateInput): string {
    const parts = [
        t.title,
        t.description,
        t.frameworks,
        t.use_cases,
        t.css,
        t.databases,
        t.authentication,
        t.cms,
    ];
    if (t.readme_text) parts.push(t.readme_text.slice(0, 2000));
    return parts.filter(Boolean).join(" ").trim();
}

function hashCode(s: string): number {
    let h = 0;
    for (let i = 0; i < s.length; i++) {
        h = (Math.imul(31, h) + s.charCodeAt(i)) | 0;
    }
    return h;
}

function mulberry32(a: number) {
    return function () {
        a |= 0;
        a = (a + 0x6d2b79f5) | 0;
        let t = Math.imul(a ^ (a >>> 15), 1 | a);
        t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t;
        return ((t ^ (t >>> 14)) >>> 0) / 4294967296;
    };
}
```

```typescript
import { default as sqliteVec } from "sqlite-vec";

export class TemplateDatabase {
    // ... existing ...

    init() {
        // existing templates/search tables
        this.db.exec(`
            CREATE VIRTUAL TABLE IF NOT EXISTS embeddings USING vec0(embedding float[384])
        `);
    }

    reset() {
        this.db.exec("DROP TABLE IF EXISTS embeddings");
        // existing drops
        this.init();
    }

    saveEmbeddings(rows: { rowid: number; embedding: Float32Array }[]) {
        const insert = this.db.prepare(
            "INSERT OR REPLACE INTO embeddings (rowid, embedding) VALUES (?, ?)"
        );
        const save = this.db.transaction((data: typeof rows) => {
            for (const d of data) insert.run(d.rowid, d.embedding.buffer);
        });
        save(rows);
    }

    semanticSearch(queryEmbedding: Float32Array, limit: number): Template[] {
        return this.db
            .prepare(
                `
                SELECT t.*, distance FROM embeddings e
                JOIN templates t ON t.id = e.rowid
                WHERE e.embedding MATCH ?
                ORDER BY distance
                LIMIT ?
                `
            )
            .all(Buffer.from(queryEmbedding.buffer), limit) as Template[];
    }
}
```

## Open questions / risks

1. **sqlite-vec exact API** — verify the vector insertion and query syntax in the installed Python package before finalizing SQL.
2. **Windows compatibility** — `sentence-transformers` and PyTorch may be heavy on Windows; the `semantic` extra is optional, but ensure CI covers both with and without it.
3. **TypeScript sqlite-vec** — verify Node binding works on Windows/WSL and in Docker build.
4. **Model download size** — first-time use downloads ~80 MB; document this clearly.
5. **No free lunch on intent** — if the embedding text is poor, results will be poor. Ensure `readme_text` is included up to a limit.
